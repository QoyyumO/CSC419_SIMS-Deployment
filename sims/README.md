## Installation

### Prerequisites

To get started with SIMS, ensure you have the following prerequisites installed and set up:

- [Node.js 24.x.x](https://nodejs.org/en/about/previous-releases) or later

### Steps

- Clone the repository
- Navigate to the project on your terminal
- Install dependencies:
  ```bash
  npm install
  ```
  - Set up secrets for the Next frontend server(run from the root directory):

  ```bash
  cp ./.env.example ./.env.local
  ```
- Start the frontend Next server:
  ```bash
  npm run dev
  ```
